
[[useEffect]]内で張ったイベントやタイマーを解除してバグやメモリリークを防ぐ処理

まぁシンプルにいっぺんリセットするってこと。
下に一例

```tsx
useEffect(() => {
  // ① なにか副作用の処理（イベント・タイマーなど）
  const timerId = setInterval(() => {
    console.log("tick")
  }, 1000)

  // ② クリーンアップ関数（戻り値として return する関数）
  return () => {
    clearInterval(timerId) // ← ①を片付ける
  }
}, [])

```
